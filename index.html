About

The Canvas API provides a means for drawing graphics via JavaScript and the HTML <canvas> element. Among other things, it can be used for animation, game graphics, data visualization, photo manipulation, and real-time video processing.

What we can draw on the canvas

  • Line
  • Rectangle
  • Circle
  • Text
  • Image
  • and many more

Canvas Gradients

Gradients can be used to fill rectangles, circles, lines and text. Shapes on the canvas are not limited to solid colors.

There are two types of gradients: linear and radial.

  • createLinearGradient(x0,y0,x1,y1) - Creates a linear gradient
  • createRadialGradient(x0,y0,r0,x1,y1,r1) - Creates a radial gradient
